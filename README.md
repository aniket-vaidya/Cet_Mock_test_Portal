📘 Online Test & Performance Analysis System

📝 Description

This project is a web-based Online Test and Performance Analysis System built using PHP and MySQL. It allows users to register, log in, attempt tests, and receive detailed performance insights including accuracy, section-wise analysis, and visual reports.

The system helps users identify their strengths and weaknesses, making it ideal for exam preparation and learning improvement.

🚀 Features
🔐 User Authentication (Login/Register)
👥 Role-based Access (Admin & User)
📝 Online Test System
📊 Accuracy & Section-wise Analysis
📈 Interactive Charts (Chart.js)
📘 Detailed Question Solutions
📄 PDF Report Download
🔑 Session Management
🛠️ Tech Stack
Technology	Usage

Chart.js	Data Visualization
